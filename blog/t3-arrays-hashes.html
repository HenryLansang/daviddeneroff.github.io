Hashes and Arrays

10/26/14


Using hashes and arrays to store collections of information is essential when dealing with a large amount of information. While there are a lot of similarities between hashes and arrays, there are also distinct differences that are important to understand.
Here we have an example of an array:

my_array=[“here”, “it”, “is”]

The information stored in the array can be called upon by referring to the corresponding index number.
my_array[0] brings us to the value “here” in the array.
Hashes also use key-value pairs, but in a much different way than arrays.

great_hash = {
“key” => “value”
}

Within a hash containing a lot of information, theirs is no need to remember where in the order it lies, like you would have to with an array. Instead, simply remember the key to find the value associated with it.
